Title: Gestione Bilancio

How to execute:

ON LINUX OS:
- cd <path-to-project>/src/
- <path-to-javac>/javac -d ../bin/ -cp ../lib/*:. vociBilancio/GestioneBilancio.java
- cd ../bin/
- java -cp ../lib/*:. vociBilancio.GestioneBilancio

ON WINDOWS OS:
- cd <path-to-project>\src\
- "<path-to-javac>\javac.exe" -d ..\bin -cp ..\lib\*;. vociBilancio\GestioneClasse.java
- cd ..\bin
- java -cp ..\lib\*;. vociBilancio.GestioneBilancio
